# crm_app
Mini CRM for calling


Установка и запуск
Создайте виртуальное окружение:

bash
```
python -m venv venv
source venv/bin/activate  # Linux/Mac
```
# или
```
venv\Scripts\activate  # Windows
```
Установите зависимости:

bash
```
pip install flask
```
Запустите приложение:

bash
```
python app.py
```
Основные функции
Загрузка номеров: Массовая загрузка телефонов списком

Автоматическая валидация: Проверка и форматирование номеров

Управление клиентами: Редактирование всех полей

Статусы: Система статусов с фильтрацией

Статистика: Показывает количество клиентов по статусам

Поиск/фильтрация: По статусам
